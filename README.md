# silly willy hackpad

a 6-key macropad with a 0.91" OLED display and a rotary encoder. everything is controlled with KMK firmware.

im very new to electronics and hardware and this was a thing for me

# BOM

1x XIAO RP2040
6x MX-like switches
6x DSA keycaps
6x through-hole 1N4148 diodes
1x SK6812 MINI-E LED
1x 0.91" 128x32 OLED display
1x EC11E rotary encoder
1x case, made of 2 printed parts (top.step and bottom.step )
4x M3x16mm screws
